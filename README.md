## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- WordPress Installed

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Aditya9056/a-wordpress-theme
   ```

2. Copy the theme into the theme folder in `wordpress-content/themes` folder.
